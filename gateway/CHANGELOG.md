# Changelog

本子项目变更. 写作规范见 [deploy.md §CHANGELOG 写作](../deploy.md#changelog-写作).

## [1.10.0] - 2026-05-20

### Added

- 公网入口接入 Google Workspace 单点登录, 仅 `@elestyle.jp` 员工可访问业务页面.
- 顶栏显示当前登录身份, 提供一键登出入口.

## [1.9.9] - 2026-05-20

### Fixed

- 单张截图损坏不再让整次任务卡在"运行中".
- 本地 agent 失联或卡住时给出明确超时提示, 不再无限转圈.

## [1.9.8] - 2026-05-20

### Fixed

- 任务异常重发结果时不再覆盖已完成的最终状态.

## [1.9.7] - 2026-05-20

### Fixed

- 任务耗时统计修正, 不再把派发等待时间算进执行时长.
- 网关异常给出明确错误提示, 不再只显示无文案的服务端错误.

## [1.9.6] - 2026-05-20

### Fixed

- 任务列表表头点击排序恢复正常.

## [1.9.5] - 2026-05-20

### Fixed

- 任务部分成功 + 部分未跑的中间态不再被错误标记为"失败".

## [1.9.4] - 2026-05-20

### Security

- 收紧上游代理, 屏蔽内网探测与上游凭据回写.
- 限制单张截图体积上限.

## [1.9.3] - 2026-05-20
